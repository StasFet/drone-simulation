## Drone Simulation
This is a side project I'm doing to improve my understanding of control systems (in particular cascaded PIDs) as well as pathing while picking up some Unity and C# along with it. Here is a little list of the features/cool parts of this project:
- Cascaded PID system
- Quaternion-powered attitude control (yaw WIP at the moment)
- Three manual control modes (rotation, velocity, position)
- Path generation with Catmull-Rom splines
- Gaussian eliminator for 4 equations (not too crazy but I'm proud of it)
